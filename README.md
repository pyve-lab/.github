# 2025-2026 Backend Developer Competency Roadmap

**Python Core · Computer Science · Linux System · Information Processing Engineer**

본 문서는 Java 백엔드 개발자 팀의 기초 역량 강화(CS) 및 국가기술자격(리눅스마스터 2급, 정보처리기사) 취득을 위한 장기 학습 로드맵입니다.

## 1. 개요 (Overview)

* **기간:** 2025년 11월 10일 ~ 2026년 4월 19일 (약 23주)
* **참여:** Java Backend Developers (5명)
* **핵심 목표:**
    1.  **Python 활용:** 실무 스크립팅 및 운영체제 실습을 위한 언어 도구 확보
    2.  **CS 심화:** 컴퓨터 구조 및 운영체제 원리 이해 (가상화, 스레드, 동시성 등)
    3.  **자격 증명:** 리눅스마스터 2급 및 정보처리기사 실기 합격

## 6. 운영 규칙 (Rules)

1.  **Repository:** 모든 학습 코드와 정리는 개인 브랜치에서 작업합니다.
2.  **Study Meeting:** 정해진 시간에 오프라인/온라인으로 모여, 학습한 내용 중 **실무와 연결되는 지점**을 중심으로 토론합니다.
3.  **Assignments:**
    * **Python/CS Phase:** 해당 주차의 핵심 개념을 요약 정리하여 커밋.
    * **Exam Phase:** 기출문제 풀이 인증 및 오답 노트 공유.
